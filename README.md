## Edit field during review (by Ankicraft)

You're reviewing a card and notice a typo. You stop, open the note editor, find the field, fix it, close — and by now you've completely lost your focus. There has to be a better way. 🖊️

Edit field during review gives Anki the inline editing it's always been missing.

* 🖱️ **Click any text on the card to edit it in place** — no extra windows, no interruptions to your review session.
* ✏️ **Works on every field and every note type** — basic, cloze, reverse, and any custom template.
* 🏷️ **Edit tags too** — click the tag area, type the tags separated by spaces, done.
* 🔵 **Cloze-safe editing** — clicking a cloze field shows its raw syntax (`{{c1::...}}`), so you can fix it without breaking the deletion.
* 💾 **Saves automatically** when you click away, press Enter, flip the card, or move to the next one — no confirmation needed.
* ↩️ **Fully undoable** — every change goes through Anki's undo system. **Ctrl+Z** brings it back.
* ⚡ **No flicker, no reload** — the card never redraws itself; only the edited region updates.

## HOW TO USE

1. Install the add-on and restart Anki.
2. Start a review session as usual.
3. Click on any text you want to change — a blue outline marks the editable area.
4. Type your correction and click away (or press Enter) to save. Press Escape to also save and exit.

## SETTINGS

No configuration needed. The add-on works out of the box on every note type without touching your templates.

## COMPATIBILITY

Tested on Anki 25.09.5 (Windows 11). Should work on macOS and Linux — feedback from other platforms is welcome.
